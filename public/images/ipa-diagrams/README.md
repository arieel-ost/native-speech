# IPA Vocal Tract Diagrams

Reference images scraped from thesoundofenglish.org interactive IPA chart.
These are for reference only — to create our own custom diagrams.

## Image Types

- **Monophthongs**: Each has 2 images — mouth cross-section (`-mouth`) + vocal cord (`-vocal`)
- **Diphthongs**: Animated GIFs showing tongue movement transition
- **Consonants**: Single vocal tract cross-section, grouped by **place of articulation**
