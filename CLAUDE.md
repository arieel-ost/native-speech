# Native Speech

## Package Manager

- **bun** — use `bun` for all installs, scripts, and dependency management
- Run scripts: `bun dev`, `bun build`, `bun start`
- Install deps: `bun add <pkg>`, `bun add -d <pkg>`
- Lockfile: `bun.lock` (never commit a pnpm or npm lockfile)

## Stack

- Next.js 16
- React 19
- TypeScript
- next-auth v5 (beta)
